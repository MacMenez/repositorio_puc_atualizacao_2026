# 📖 Sobre o Curso

A Engenharia de Computação é uma graduação que integra conhecimentos de:

- 💻 Computação
- ⚡ Eletrônica e Sistemas Embarcados
- 📊 Matemática e Estatística
- 🤖 Inteligência Artificial

Desde o início da graduação, o foco é desenvolver uma base sólida para atuação acadêmica e profissional.
